# NoNet Pay

Offline UPI payment app using USSD (no internet required).

---

## Overview

NoNet Pay enables sending and receiving money without internet using USSD (*99#).  
It simplifies the complex USSD process with a clean and user-friendly interface.

## Key Features

- Send & request money via:
  - UPI ID
  - Mobile number  
- QR code scanning for payments  
- Automatic USSD handling (reduces steps)  
- Works without mobile data or WiFi  

---

## Requirements

- Android 7.0+
- UPI-enabled bank account
- Registered mobile number
- Cellular network (no internet required)
- UPI PIN

---

## Permissions

- Accessibility (required for USSD automation)
- Camera (QR scanning)
- Vibration (optional)

---

## Installation

1. Download APK  
2. Enable **Install from Unknown Sources**  
3. Install and open the app  
4. Enable **Accessibility Service**  

---

## Usage

### Send Money
- Enter UPI ID or mobile number  
- Enter amount  
- Confirm with UPI PIN  

### Request Money
- Enter amount  
- Use UPI ID or mobile number  
- Confirm with UPI PIN  

### QR Payment
- Scan QR  
- Verify details  
- Enter UPI PIN  

---

## Troubleshooting

- Ensure Accessibility Service is enabled  
- Grant call permissions for USSD  
- Enable camera permission for QR  
- Maintain stable cellular signal  

---

## Security

- No transaction data stored  
- No internet usage  
- UPI PIN handled securely by bank  
- Follows standard UPI security protocols  

---

## ⚠️ Limitations

- Requires cellular network (USSD works on 2G/3G)  
- ✅ Works best with Airtel SIM  
- ❌ Not supported on Jio (4G/5G-only network, no USSD support)  
- Slower than online UPI (5–15 seconds)  
- Limited device and bank compatibility  
- No transaction history
